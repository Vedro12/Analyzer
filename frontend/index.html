Это инструмент анализа облачных ресурсов с помощью AI-ассистента. Она помогает получать структурированные рекомендации и анализ.

Для получения персонализированных рекомендаций рекомендуется указать OAuth-токен и ID каталога.

Получение OAuth-токена

  1. Перейдите по ссылке
  2. Нажмите «Разрешить»
  3. Скопируйте полученный токен
  4. Вставьте его в поле «OAuth-токен»

Получение идентификатора каталога

  1. Откройте консоль облака
  2. Выберите нужный каталог
  3. Скопируйте идентификатор под названием каталога
  4. Вставьте его в поле «Идентификатор каталога»

Также идентификатор можно получить из URL страницы:

https://console.yandex.cloud/folders/<folder_id>

Сбор данных

Нажмите кнопку «Собрать данные», чтобы загрузить информацию о ресурсах.
