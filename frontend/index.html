Облачный ассистент — это помощник на основе искусственного интеллекта.

Ассистент отвечает на вопросы о сервисах Yandex Cloud и может проводить диагностику отдельных ресурсов для предложения персонализированных ответов.

Для проведения диагностики нужно указать OAuth-токен и идентификатор каталога. Это позволит ассистенту собрать информацию об актуальном состоянии ресурсов.

Как получить OAuth-токен

  1. Перейдите по ссылке
  2. Нажмите «Разрешить»
  3. Скопируйте токен, указанный на странице
  4. Вернитесь на страницу ассистента и вставьте токен в поле «OAuth-токен»

Как получить идентификатор каталога

  1. Откройте консоль облака
  2. Выберите нужный каталог
  3. Скопируйте идентификатор под названием каталога
  4. Вернитесь на страницу ассистента и вставьте его в поле «Идентификатор каталога»

Также идентификатор можно получить из URL страницы:

https://console.yandex.cloud/folders/<folder_id>

Как начать сбор данных

Нажмите кнопку «Собрать данные», чтобы ассистент получил актуальную информацию о ресурсах.

Готово!

Начните диалог с Облачным ассистентом.

Общение с ассистентом также доступно через расширение для Chrome.
